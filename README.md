# Gender-Inequality-Project
Tableau Link: https://public.tableau.com/views/DIDA260Midterm_17292903708480/Dashboard1?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link  

Project analyzes how gender inequality informs the marital status of young individuals in different countries and regions. 
- Findings include: Regions with a higher gender inequality index have a larger amount of males and females married by 18 years old, and there is a larger percentage of females married by 18 compared to males
